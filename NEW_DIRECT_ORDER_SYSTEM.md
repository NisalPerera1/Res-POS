## 🎯 **Complete Workflows:**

### **Workflow 1: Start New Order**
1. **Click "⚡ Direct Order"**
2. **Click "➕ Create New Direct Order"**
3. **Add items** to cart
4. **Send KOT** (optional)
5. **Click "💳 Charge"** to pay
6. **Order completed** → returns to pending list

### **Workflow 2: Continue Existing Order**
1. **Click "⚡ Direct Order"**
2. **Click on pending order** (DIR-12345)
3. **Cart loads** with all items
4. **Add more items** or **checkout**
5. **Payment processed** → order removed from pending

### **Workflow 3: Switch Between Orders**
1. **Working on Order A**
2. **Click "← Back"** (saves Order A)
3. **Click "⚡ Direct Order"**
4. **Click on Order B** (switches instantly)
5. **Work on Order B**
6. **Switch back** to Order A anytime

## 🔧 **Troubleshooting:**

### **If Pending Orders Don't Show:**
1. **Refresh page** (F5)
2. **Check console** for errors
3. **Verify network** connection

### **If Order Switch Fails:**
1. **Click order again**
2. **Check order status** (must be unpaid)
3. **Try creating new order**

### **If Cart Items Missing:**
1. **Order should auto-load** from pending list
2. **Click on order** in pending list
3. **Items will appear** in right panel
